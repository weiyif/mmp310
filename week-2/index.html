Intro to JavaScript

JavaScript is a scripting language, and is optimal for creating dynamic content on the web. It has nothing to do with the other programming language Java, but it was named after it to jump on the 'Java' hype. It was originally called LiveScript.

Think of a programming language like a recipe book or a manual. A computer doesn't interpret instructions the same way we do, so we need to be very clear about how we're describing them. When we're programming, we need to understand what our ultimate goal is, and break it down into small little parts that the computer will understand.

Comptuters create models of the world using data. Those models include many objects. In object-oriented programming, objects can be repeated over and over and consist of:

  • Properties
  • Events
  • Methods

Really utilizing object oriented programming comes later, but it's worth understanding how a computer thinks of everything.

Let's start learning some JavaScript programming

We'll start with learning about the console. In Chrome, that will be in View -> Developer -> JavaScript Console.

The shortcut is CMD + OPTION+ J.

Essentially we've overriding our browser and we can code how we want now. It's only temporary, and when we refresh the browser it disappears. Let's start with a very important function that we're going to use constantly to debug our work, called the console.log().

Start by typing:

    console.log("Hello world!");

It's not much, but everytime you get stuck with your work, you'll want to console.log a variable or function to figure out what's gone wrong.

Also note the ';' at the end of the line of code. This is standard practice in coding. It's signifying 'end of statement'. It's actually usually not necessary in JavaScript, but there can be errors and it can get confused without it, so it's good practice.

Now we'll make changes to the document. This is getting into the Document Object Model which is how we access the content and style of the page. We'll learn more about that later. For now type each of these lines in the console and press enter in between:

    document;
    document.body;
    document.body.style;
    document.body.style.color = "pink";

What we did there was go into the webpage's style and change the color of the font. We can also just re-write the entire page if desired:

    document.write("New website!");

Rather than dynamically changing a full website, it's more likely to modify some HTML. This paragraph has the ID of "aParagraph".

    var theWords = document.getElementById("aParagraph");
    theWords.innerHTML = "New Words";

This gives you an idea of what's possible with JavaScript. Anything that's been labeled by our HTML or CSS can be accessed and altered by JavaScript

var is short for 'variable' in JavaScript. Variables are used to hold information which can change. In other programming languages, you need to specify if it's a number, or a word, a boolean statement. JavaScript doesn't really care. But be careful not to get confused.

And with all other programming languages, we can do some math as well.

    var x;      //Name the variable
    x = 10;     //Assign it a value
    typeof x;   //A function to see type of a variable
    x = x + 5;
    x++;        //Handy shortcut to say x equals x plus one
    x;          //The ++ isn't stated immediately
    x = "A number";    //Now it's a String
    typeof x;
    x = "10";   //Quotes still mean string
    typeof x;
    x = false;  //This is boolean which can be true or false
    typeof x;

Comments are deciphered using // in JavaScript. Multi-line comments together can use /* and */ like in CSS.

In a code editor

Eventually, we'll want to have a separate JavaScript file just to make it cleaner and so it can more easily be applied to all of our pages. For now, we can insert JavaScript in our index.html in script tags.

    <script type="text/javascript">
        Do something here
    </script>

Buttons

A button is one of the easiest things we can add in HTML with some interactivity.

Add this to your HTML:

    <button id="submit">Click me.</button>

Then add this as a script in your code.

    var submit = document.getElementById("submit");
    
    submit.onclick = function() {
	    console.log("Button pushed!");
    };

Or add an alert to it:

    alert("Congrats you win!");
